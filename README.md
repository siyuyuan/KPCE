# KPCE

Code for our ACL 2023 paper: Causality-aware Concept Extraction based on Knowledge-guided Prompting.

## Requirements
- torch == 1.4.0
- transformers == 4.2.0

## Dataset
sample dataset is release on dataset/

- Chinese data: abstract \t 0/1 strings for the start position \t 0/1 strings for the end position
